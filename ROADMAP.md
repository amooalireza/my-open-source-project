# Roadmap

This roadmap shows the plan for building this project from a beginner learning space into a real portfolio for store website design.

## Main Direction

The main goal is to learn how to build modern cinematic scroll websites for stores and small businesses, then use those examples to get real clients.

This project is focused on stores that do not have a website yet, sell mostly through Instagram, or already have a weak website that needs improvement.

## First Month Goals

- [x] Create the GitHub repository
- [x] Add the first README
- [x] Add contribution guidelines
- [ ] Build the first cinematic scroll website concept
- [ ] Create 2-3 sample website layouts for stores
- [ ] Prepare a simple portfolio presentation
- [ ] Talk to at least 20 stores
- [ ] Try to close 3 real website contracts
- [ ] Write down what store owners need
- [ ] Improve the project based on real feedback

## 3 Month Goals

- [ ] Improve website design skills
- [ ] Build more store website examples
- [ ] Create a small personal brand around web design
- [ ] Start social media pages to attract more clients
- [ ] Post website examples and progress online
- [ ] Find more stores that need websites
- [ ] Build a small team
- [ ] Create a simple process for working with clients
- [ ] Offer website and Instagram support packages
- [ ] Turn this project into a serious portfolio

## Website Ideas

- Cinematic landing page for a clothing store
- Product showcase website for a local shop
- Simple online store page
- Instagram-connected store website
- Digital catalog for small businesses
- Modern one-page website with smooth scrolling

## Business Goals

- Learn website design by building real examples
- Use AI tools to move faster and improve quality
- Build trust with small store owners
- Help stores improve their online presence
- Get real clients and real feedback
- Grow from a solo beginner project into a small team

## Long-Term Vision

The long-term vision is to create a small team that helps stores and small businesses sell better online through websites, Instagram, simple branding, and AI-assisted digital tools.
